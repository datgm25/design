10秒以内にトゲに当たらないようにコインを全て取るゲームです。

# 操作とルール

- マウスで操作
- 10秒以内にコインを全て取ってください
- トゲの甲羅にぶつかるか10秒経過でゲームオーバー

# 公開先

- https://datgm21.github.io/yoketoru2021-demo/
